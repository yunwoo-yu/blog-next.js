Wello에서 프론트엔드 개발자로 근무하며 사용자 서비스, Wello-biz, 통합 어드민, 웰마켙 커머스 영역의 화면 개발과 API 연동을 담당했습니다. React, Next.js, TypeScript 기반 환경에서 기능 구현, 상태 관리, 폼 검증, QA 및 운영 이슈 대응까지 수행했습니다.

주요 업무는 웰마켙 커머스 신규 구축이었습니다. 상품 홈, 상품 상세, 장바구니, 주문서, 결제 완료, 주문내역 화면을 개발하고 NICE Payments 결제, 쿠폰/포인트 적용, 배송비 및 최종 결제금액 계산, 주문 취소, 반품, 교환, 구매확정 플로우를 구현했습니다.

어드민에서는 입점업체, 상품, 주문, 후기/문의, 쿠폰, 웰포인트, 오늘마감 특가, 이벤트/기획전 관리 화면을 개발했습니다. 운영자와 입점업체 권한에 따라 접근 가능한 기능이 달라지도록 화면과 API 흐름을 구성했습니다.

그 외 Wello-biz 상세검색 키워드 입력 UX 개선, Wello PC 홈·통합검색·정책 상세 개편, 고향사랑기부 중복 기부 방지 및 외부 납부 상태 확인 플로우 개선, 마이데이터 인증·동의·전송요구·철회 화면 개발을 수행했습니다.
