(주)살린에서 프론트엔드 개발자로 근무하며 AI 챗봇 플랫폼, WebGL 아바타 서비스, 스타트업 업무 지원 서비스, 관리자 CMS의 화면 개발과 API 연동을 담당했습니다. React, TypeScript, Next.js, TanStack Query, Emotion, Vite, Webpack 기반 환경에서 서비스 웹과 CMS 기능 구현, 상태 관리, 폼 처리, 배포 및 운영 이슈 대응을 수행했습니다.

주요 업무는 AI Talker 챗봇 서비스와 관리자 CMS 구축이었습니다. OpenAI Assistant API 기반 채팅 처리를 화면에 연결하고, 실시간 답변 생성, 채팅 히스토리 조회, 역방향 무한 스크롤, 모바일 채팅 UX를 구현했습니다. CMS에서는 프롬프트, 모델, 응답 스타일, 파일 등록, 벡터 스토어, 통계 대시보드, AI 콘텐츠 자동 생성 기능을 개발해 고객사가 챗봇을 직접 운영할 수 있도록 구성했습니다.

또한 CSR 기반 React 서비스를 Next.js SSR 구조로 전환하고, 고객사별 동적 메타데이터, 다국어 OpenGraph, API 기반 사이트맵을 적용해 검색 노출 관리를 자동화했습니다. 무료 고객사 대상 광고 삽입 기능을 구현해 서비스 수익화 구조도 함께 반영했습니다.

Swing 프로젝트에서는 서비스 웹과 관리자 CMS 프론트엔드를 단독 개발했습니다. 인증, 대시보드, 사업지원 관리, 계약/문서 관리, WebGL 가상공간 진입 화면과 연결 로직을 구현하고, Lazy Loading, 코드 스플리팅, 번들 분석을 통해 초기 로딩 부담을 줄였습니다.

Localo WebGL 아바타 서비스에서는 초기 로딩 성능 개선과 CMS 개발을 담당했습니다. Suspense와 lazy loading으로 렌더링 블로킹 구간을 분리하고, 3D 및 JSON 정적 자산을 S3/CDN으로 이전해 첫 화면 표시 속도를 개선했습니다. 레거시 대형 컴포넌트는 공통 컴포넌트와 커스텀 훅으로 분리해 유지보수성을 높였습니다.

그 외 CMS 전용 UI Kit과 서비스/CMS 보일러플레이트를 구축했습니다. Rollup 기반 UI 라이브러리, Storybook, Chromatic, Yarn Berry 모노레포, Lerna 버전 관리 구조를 도입해 신규 CMS 프로젝트의 초기 세팅과 공통 UI 재사용 효율을 높였습니다.
